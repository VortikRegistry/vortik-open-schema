# Candidate backlog

## Purpose

This backlog tracks public technical candidate terms that may deserve monitoring but are not registry entries.

## Required disclaimers

- Candidate does not mean registry entry.
- Candidate does not mean ENS anchor.
- Candidate does not mean official Ethereum status.
- Candidate does not mean promotion.
- Candidate does not mean commercial opportunity.
- Future registry changes require separate pull requests and primary-source review.

## Boundary

This backlog is not a registry promotion queue. Any future registry addition, schema change, status update, classification change, type change, or ENS anchor addition must happen in a separate pull request with primary-source review.
