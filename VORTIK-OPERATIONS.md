## Main Mistakes To Avoid

| Mistake | Why it is wrong |
|---|---|
| Editing `docs/market.html` directly | It is regenerated by pipeline |
| Editing derived files manually | They can be overwritten |
| Treating `index.md` as live home | Live home is served from `/docs` |
| Forgetting to check Actions after push | Can hide broken generation |
| Updating public copy in the wrong file | Changes may not persist |
