# LEGACY — Coordination Pipeline (Deprecated Model)

This file represents an outdated linear pipeline model of Ethereum block production.

The current protocol design (post-ePBS / FOCIL) is not a strict pipeline. It is better described as a mechanism system with:
- protocol-defined roles
- commitments
- constraints
- parallel validation and enforcement processes

This file is preserved only for historical reference.
